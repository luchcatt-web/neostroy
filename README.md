# НЕОстрой — Сайт строительной компании

Современный одностраничный сайт для строительной компании НЕОстрой в Брянске, выполненный в стиле премиум дизайна.

## Структура проекта

- `index.html` — основная HTML структура сайта
- `styles.css` — стили в премиум дизайне
- `script.js` — JavaScript для интерактивности
- `README.md` — документация

## Особенности

- ✅ Современный премиум дизайн в стиле designleo.ru
- ✅ Адаптивная верстка для всех устройств
- ✅ Плавная прокрутка и анимации
- ✅ Модальные окна для форм обратной связи
- ✅ Мобильное меню
- ✅ Форматирование телефонных номеров
- ✅ Валидация форм

## Секции сайта

1. **Главная (Hero)** — приветственный блок с призывом к действию
2. **Почему выбирают нас** — преимущества компании
3. **Услуги** — детальное описание услуг:
   - Дизайн и проектирование
   - Строительство и реализация
   - Управление и комплектация
4. **Статистика** — достижения компании
5. **О нас** — история компании
6. **Портфолио** — примеры выполненных проектов
7. **Контакты** — контактная информация и форма обратной связи

## Запуск

Просто откройте `index.html` в браузере. Для локальной разработки можно использовать любой веб-сервер:

```bash
# Python 3
python -m http.server 8000

# Node.js (с http-server)
npx http-server

# PHP
php -S localhost:8000
```

Затем откройте `http://localhost:8000` в браузере.

## Настройка

### Изменение контента

Все текстовые данные находятся в `index.html`. Вы можете легко изменить:
- Название компании
- Контактную информацию
- Описание услуг
- Портфолио проектов
- Статистику

### Изменение цветов

Цветовая схема определена в `styles.css` в секции `:root`:

```css
:root {
    --primary-color: #1a1a1a;
    --secondary-color: #f5f5f5;
    --accent-color: #d4af37;
    /* ... */
}
```

### Интеграция форм

В `script.js` формы обрабатываются локально. Для реальной отправки данных на сервер:

1. Создайте backend endpoint для обработки форм
2. Замените обработчики в `script.js` на реальные AJAX запросы
3. Добавьте обработку ошибок и успешных отправок
